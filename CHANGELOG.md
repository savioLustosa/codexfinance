# Changelog

## [0.2.0] - 2026-03-14

### Added
- Workflow de CI no GitHub Actions para instalar dependências e executar lint, typecheck, testes e build.
- Script `release:check` no pacote raiz para validar release localmente antes de subir para GitHub.
- Estrutura de versionamento explícita em todos os pacotes do monorepo.

### Changed
- Bump de versão do monorepo e workspaces para `0.2.0` para início de testes.
